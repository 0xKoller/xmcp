# Getting Started

Welcome to the Getting Started guide for xmcp! This section will walk you through everything you need to know to start building with xmcp.

## What You'll Learn

In this section, you'll learn how to:

- Install xmcp and set up your development environment
- Create your first xmcp project
- Build your first tool
- Understand the project structure
- Deploy your application

## Prerequisites

Before getting started, make sure you have:

- Node.js 20+ installed
- A text editor or IDE
- Basic knowledge of TypeScript
